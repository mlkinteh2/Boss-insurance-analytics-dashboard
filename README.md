# BOSS Insurance Dashboard — Power BI Project

## Project Overview

This project presents an interactive **Insurance Analytics Dashboard** built using Microsoft Power BI.

The dashboard analyzes policy performance, premium revenue, claim trends, and customer demographics for a fictional insurance company: **BOSS INSURANCE PVT. LTD.**

The goal is to transform raw insurance data into actionable business insights through interactive visualization.

---

## Business Objectives

Insurance companies must continuously monitor performance and risk.  
This dashboard helps stakeholders:

- Track total premium revenue
- Analyze claim payouts
- Compare policy types
- Monitor claim statuses
- Understand customer demographics
- Evaluate policy activity levels

---

## 🗂 Dataset Description

The dataset includes the following attributes:

| Column | Description |
|--------|------------|
| PolicyNumber | Unique policy identifier |
| CustomerID | Unique customer ID |
| Gender | Customer gender |
| Age | Customer age |
| PolicyType | Type of insurance (Travel, Health, Auto, Life, Home) |
| PolicyStartDate | Policy start date |
| PolicyEndDate | Policy end date |
| PremiumAmount | Premium paid by customer |
| CoverageAmount | Maximum insured amount |
| ClaimNumber | Claim identifier |
| ClaimDate | Date of claim |
| ClaimAmount | Amount claimed |
| ClaimStatus | Pending / Rejected / Settled |

---

## Key Performance Indicators (KPIs)

### Total Premium Amount
Total revenue collected from all policies.

### Total Coverage Amount
Total insured value across policies.

###  Total Claim Amount
Total amount claimed by policyholders.

###  Claim Status Distribution
Breakdown of claims into Pending, Rejected, and Settled.

---

## Dashboard Components

###  KPI Overview
Displays high-level metrics:

- Total Premium Amount
- Total Coverage Amount
- Total Claim Amount

---

###  Policy Type Analysis
Bar chart comparing premium distribution across policy types to identify top-performing categories.

---

###  Claim Status Analysis
Visualization showing distribution of claims by status:

- Pending
- Rejected
- Settled

---

### Coverage by Age Group
Analyzes how coverage varies across different age segments.

---

###  Active vs Inactive Policies
Donut chart showing the proportion of active and inactive policies.

---

###  Gender Distribution
Displays the number of male and female policyholders.

---

### 🔹 Interactive Filters (Slicers)

Users can filter the dashboard using:

- Policy Number
- Claim Number
- Customer ID

All visuals update dynamically based on selected filters.

## 📈 Key Insights

- Travel policies generate the highest premium revenue.
- A significant number of claims remain pending.
- Coverage amounts vary across age groups.
- Gender distribution among customers is relatively balanced.
- Policy activity levels indicate both active and inactive portfolios.
